## Installation

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/urooba01/mywebsite.git
    ```
2. Navigate to the project directory:
    ```bash
    cd mywebsite
    ```
3. Open `index.html` in your browser.
